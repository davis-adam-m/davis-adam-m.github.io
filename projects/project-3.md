## Project Description
* Used SQL DDL queries to create a database with dimension/fact tables, assign column names/data types, establish primary/foreign key relationships, and populate data.
* Created stored procedures/views/triggers to allow upstream database input and querying while maintaining data integrity and security. 
* Executed basic and advanced DML queries such as filtering/aggregation and complex multi-table joins for accurate answers to routine business questions on sales metrics and product performance.
